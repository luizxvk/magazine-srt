# 📋 BRIEFING - Materiais Visuais para Vídeo Promocional ROVEX

> **Objetivo:** Solicitar à IA imagens/capturas das páginas da Rovex para montar o vídeo promocional

---

## 🎯 O QUE PRECISAMOS

Estamos criando um vídeo promocional de **2 minutos** para a plataforma **Rovex** - uma ferramenta SaaS que permite criar comunidades gamificadas personalizadas.

Precisamos de **capturas de tela / mockups / visuais** das seguintes páginas e elementos:

---

## 📸 LISTA DE PÁGINAS/ELEMENTOS NECESSÁRIOS

### 1️⃣ TELA DE LOGIN / REGISTRO
- Página de login estilizada
- Formulário de cadastro
- Logo da plataforma em destaque

### 2️⃣ FEED PRINCIPAL
- Feed com posts (fotos e vídeos)
- Curtidas e comentários visíveis
- Layout estilo rede social moderna

### 3️⃣ STORIES
- Área de stories no topo
- Visualização de um story aberto
- Indicador de stories não vistos

### 4️⃣ PERFIL DO USUÁRIO
- Página de perfil completa
- Avatar, bio, estatísticas
- Badges/conquistas visíveis
- Barra de XP/nível
- Personalização (cores, bordas, fundos)

### 5️⃣ SISTEMA DE GAMIFICAÇÃO
- Tela de níveis e XP
- Lista de conquistas/badges
- Ranking de usuários (leaderboard)
- Modal de "subiu de nível"
- Bônus diário / login streak

### 6️⃣ ECONOMIA VIRTUAL (ZIONS)
- Saldo de moedas virtuais
- Histórico de transações
- Modal de compra de Zions
- Tela de saque (PIX)

### 7️⃣ LOJA / MERCADO
- Loja de produtos (keys, gift cards)
- Mercado entre usuários
- Tela de compra/checkout
- Loja de customizações (fundos, cores, badges)

### 8️⃣ GRUPOS E CHAT
- Lista de grupos
- Chat de grupo aberto
- Chat privado (mensagens diretas)
- Criação de grupo

### 9️⃣ PERSONALIZAÇÃO
- Tela de customização de perfil
- Temas animados disponíveis
- Seleção de cores de destaque
- Bordas de perfil

### 🔟 PAINEL ADMIN
- Dashboard administrativo
- Estatísticas da comunidade
- Gestão de usuários
- Configurações da plataforma

### 1️⃣1️⃣ EXTRAS
- Rádio integrado (player)
- Catálogo de fotos
- Notificações
- Modal de eventos
- Enquetes

## 🎬 CONTEXTO DO VÍDEO

O vídeo vai seguir este roteiro:

1. **Hook** (0-5s) - Logo Rovex
2. **Problema** (5-15s) - Dificuldade de criar comunidades
3. **Solução** (15-30s) - Apresentação da Rovex
4. **Gamificação** (30-50s) - XP, níveis, badges, ranking
5. **Economia** (50-65s) - Moedas, loja, saques
6. **Personalização** (65-80s) - Temas, cores, customização
7. **Features** (80-100s) - Feed, stories, grupos, chat
8. **Casos de uso** (100-115s) - Diferentes nichos
9. **CTA** (115-120s) - Chamada para ação
